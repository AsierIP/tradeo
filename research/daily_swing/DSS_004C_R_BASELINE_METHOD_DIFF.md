# DSS-004C-R Baseline Method Diff

Probable cause: DSS-004C used a coarse per-year sampler that divided total yearly base trades across all symbols; DSS-004C-A used exact base symbol-year counts. The sample definition changed, so matched baselines are not comparable.

DSS-004C and DSS-004C-A used the same signal timing, costs, holding period, and baseline names, but they did not use the same matched-sample definition.
